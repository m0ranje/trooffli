# Портфолио Трофимовой Юли

Простой одностраничный сайт-портфолио в стиле бумажного стикер-коллажа.

## Запуск

Откройте файл `index.html` в браузере.

## Где менять данные

- Разметка и тексты: `index.html`
- Цвета, адаптивность и анимации: `styles.css`

## Стек

Обычные HTML и CSS. Next.js, npm и сборка не нужны.
